##快速安裝

本機安裝 Docker 請利用 Docker Toolbox 。

安裝網址：

Docker office site 。 [Link](https://www.docker.com/toolbox)


安裝完畢，工具包幫你安裝了以下服務：


- Docker Client
- Docker Machine
- Docker Compose
- Docker Kitematic
- VirtualBox 5.0
- Delivery Format
